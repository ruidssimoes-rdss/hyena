## Installation

```bash
npx r-ui add combobox
```

## Usage

```tsx
import {
  Combobox,
  ComboboxTrigger,
  ComboboxContent,
  ComboboxInput,
  ComboboxList,
  ComboboxEmpty,
  ComboboxGroup,
  ComboboxItem,
} from '@r-ui/react-native'

export default function MyComponent() {
  const [value, setValue] = useState('')

  return (
    <Combobox value={value} onValueChange={setValue}>
      <ComboboxTrigger placeholder="Select option..." />
      <ComboboxContent>
        <ComboboxInput placeholder="Search..." />
        <ComboboxList>
          <ComboboxEmpty>No results found.</ComboboxEmpty>
          <ComboboxItem value="option1">Option 1</ComboboxItem>
          <ComboboxItem value="option2">Option 2</ComboboxItem>
          <ComboboxItem value="option3">Option 3</ComboboxItem>
        </ComboboxList>
      </ComboboxContent>
    </Combobox>
  )
}
```

## Features

- **Search filtering**: Type to filter options instantly
- **Keyboard navigation**: ↑↓ to navigate, Enter to select, Esc to close
- **Groups**: Organize options with headings
- **Empty state**: Show message when no options match
- **Accessible**: Full ARIA combobox pattern support

## Props

### Combobox

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `string` | - | Controlled value |
| `onValueChange` | `(value: string) => void` | - | Called when value changes |
| `defaultValue` | `string` | `''` | Default value (uncontrolled) |
| `disabled` | `boolean` | `false` | Disable the combobox |
| `open` | `boolean` | - | Controlled open state |
| `onOpenChange` | `(open: boolean) => void` | - | Called when open state changes |
| `filter` | `(value: string, search: string, keywords?: string[]) => boolean` | - | Custom filter function |

### ComboboxTrigger

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `placeholder` | `string` | `'Select...'` | Placeholder text |
| `style` | `ViewStyle` | - | Additional styles |

### ComboboxInput

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `placeholder` | `string` | `'Search...'` | Search input placeholder |
| `autoFocus` | `boolean` | `true` | Focus on open |
| `showClear` | `boolean` | `true` | Show clear button |

### ComboboxItem

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `string` | Required | Item value |
| `disabled` | `boolean` | `false` | Disable this item |
| `keywords` | `string[]` | - | Additional search keywords |

### ComboboxGroup

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `heading` | `string` | - | Group heading text |

### ComboboxEmpty

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | `'No results found.'` | Empty state message |
